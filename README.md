# Telecom Traffic Forecasting (Linear Regression, XGBoost, LSTM)

Forecasting telecom **Arrivals** and **Departures** using three models:

1. Linear Regression (Moore–Penrose pseudo-inverse)
2. Autoregressive XGBoost (lag features + recursive forecasting)
3. LSTM (sequence modeling)

![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

---

## Overview

This repository compares three forecasting approaches on a real telecom traffic dataset.  
Each notebook is **self-contained**, and produces **100-step forecasts** for both Arrivals and Departures.

---

## Dataset

- **File:** `telecommunication_traffic_data.csv`  
  Contains time-related features, pricing information, and traffic signals (Arrivals/Departures).

---

## Repository Contents

### Notebooks

- `MATH_Linear Reg.ipynb`  
  Linear regression using the pseudo-inverse, includes training, evaluation, and 100-step forecasting.

- `MATH_xgboost.ipynb`  
  Autoregressive boosting model using lag features and recursive forecasting.

- `MATH_LSTM.ipynb`  
  LSTM forecasting with sequence preparation, training, and loss/forecast analysis.

### Forecast Outputs (100 periods)

- `forecast_100_periods_Linear_Reg.csv`
- `forecast_100_periods_xgboost.csv`
- `forecast_100_periods_LSTM.csv`

Each file includes 100-step forecasts for **Arrivals** and **Departures** generated by the corresponding model.

## How to Run

### 1) Clone the repository

```bash
git clone https://github.com/Zahid-Alabadllah/Telecommunication-Traffic-Forecasting.git
cd telecom-traffic-forecasting
```

### 2) Install dependencies

```bash
pip install numpy pandas matplotlib scikit-learn xgboost tensorflow
```

(If your LSTM notebook uses PyTorch, install PyTorch instead of TensorFlow.)

### 3) Run the notebooks

```bash
jupyter notebook
```

Recommended order:

1. `MATH_Linear Reg.ipynb`
2. `MATH_xgboost.ipynb`
3. `MATH_LSTM.ipynb`

Each notebook will:

- Load the dataset
- Prepare features
- Train the model
- Evaluate performance
- Generate and export 100-step forecasts

---

## Results Summary

- **Linear Regression:** stable, interpretable baseline.
- **XGBoost:** strong pattern capture, possible overfitting.
- **LSTM:** captures trends, struggles with sharp fluctuations.
